Na tela “Selecionar Variações”, os produtos não estão sendo carregados corretamente.
Ao abrir essa tela, o sistema deve "LISTA" todos os produtos pertencentes à categoria definida no Tipo de Variação.

Após o usuário selecionar as variações, o sistema deve:

Concatenar os nomes dos produtos selecionados, adicionando o prefixo “meio” antes de cada item.
Exemplo de nome final:
“meio porção de calabresa / meio porção de batata”.

Utilizar o valor definido no Tipo de Variação, e não o valor individual de cada produto.

Exibir os produtos selecionados um abaixo do outro, facilitando a visualização dos nomes.

Colocar um scroll na tela para pode aparecer o botao de confirma

Enviar essas informações também para a tela “Modo Tablet”, garantindo que o setor responsável saiba exatamente o que deve ser preparado.
Estas implentacoes deve ser feita tanto para o desktop como no telefone
